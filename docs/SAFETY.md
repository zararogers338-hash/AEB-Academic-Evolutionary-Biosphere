# Safety and Interpretation / 安全与解释边界

AEB is a research and visualization prototype.

AEB 是一个研究与可视化原型。

It should not be used as:

- an absolute academic ranking system;
- a proof of scientific truth;
- a replacement for peer review;
- an author, institution, or paper evaluation authority.

它不应被用作：

- 绝对学术排名系统；
- 科学真理证明器；
- 同行评议替代品；
- 作者、机构或论文评价权威。

The biological metaphors are useful for thinking, but they can also mislead if treated literally.

生物学隐喻有助于思考，但如果被字面化使用，也可能误导。
