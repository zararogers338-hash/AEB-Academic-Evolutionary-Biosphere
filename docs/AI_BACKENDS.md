# AI Backends / AI 后端

AEB supports optional AI interpretation.

AEB 支持可选 AI 解释。

## GGUF / llama-cpp-python

Use local `.gguf` models. This may require suitable CPU/GPU resources and compatible compilation.

## Ollama

Start Ollama first:

```bash
ollama serve
```

Then select the model in the sidebar.

## OpenAI-compatible API

Configure base URL, API key, and model name in the sidebar.

Do not send private documents to external APIs unless you understand the privacy risk.

请不要把隐私或保密文档发送到外部 API，除非你理解相关风险。
