# Data Formats / 数据格式

## Input / 输入

AEB supports:

```text
txt, pdf, md, markdown, doc, docx, json, jsonl,
csv, tsv, xlsx, xls, html, htm, xml, yaml, yml
```

## Output / 输出

AEB supports:

```text
JSON, CSV, TXT, ZIP
```

The main exported data includes total papers, total keywords, Shannon diversity, monopoly ratio, keyword table, evolution tree, and communities.

主要导出数据包括论文总数、关键词总数、Shannon 多样性、垄断占比、关键词表、进化树和群落信息。
